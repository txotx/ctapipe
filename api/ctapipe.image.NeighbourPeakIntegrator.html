NeighbourPeakIntegrator¶

Bases: ctapipe.image.charge_extractors.PeakFindingIntegrator

Charge extractor that defines an integration window defined by the peaks in the neighbouring pixels.

Parameters:
config : traitlets.loader.Config

Configuration specified by config file or cmdline arguments. Used to set traitlet values. Set to None if no configuration to pass.

tool : ctapipe.core.Tool

Tool executable that is calling this component. Passes the correct logger to the component. Set to None if no Tool to pass.

kwargs
Attributes:

List of neighbours for each pixel. Changes per telescope.
