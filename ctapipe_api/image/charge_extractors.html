Charge Extraction (cube to image)¶

extracts charge from the waveform, resulting in a single number per pixel

ctapipe.image.charge_extractors Module¶

Charge extraction algorithms to reduce the image to one value per pixel

Classes¶

ChargeExtractor([config, tool])
FullIntegrator([config, tool])
SimpleIntegrator([config, tool])
GlobalPeakIntegrator([config, tool]) Charge extractor that defines an integration window about the global peak in the image.
LocalPeakIntegrator([config, tool]) Charge extractor that defines an integration window about the local peak in each pixel.
NeighbourPeakIntegrator([config, tool]) Charge extractor that defines an integration window defined by the peaks in the neighbouring pixels.
AverageWfPeakIntegrator([config, tool]) Charge extractor that defines an integration window defined by the average waveform across all pixels.
